## Installation

To get started with the project, follow these steps:

1. Install dependencies:

```sh
npm install
```

## Available Scripts

In the project directory, you can run the following scripts:

- `npx expo start`
- `npx expo start --android`
- `npx expo start --ios`
- `npx expo start --web`
- `npm test`
